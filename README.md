# Boar Bonanza

Инди-слот **5×3** на **PixiJS 8** и **TypeScript**: барабаны, линии, скаттер, фриспины, баланс и ставка. Логика отделена от отрисовки (domain → application → presentation).

## Требования

- **Node.js** 18+ (рекомендуется LTS)
- **npm** (или совместимый менеджер пакетов)

## Установка и запуск

```bash
npm install
```

### Режим разработки

```bash
npm run dev
```

Открой в браузере адрес из терминала (обычно `http://localhost:5173`).

### Сборка для продакшена

```bash
npm run build
```

Артефакты попадут в папку `dist/`.

### Просмотр production-сборки локально

```bash
npm run preview
```

## Тесты

```bash
npm run test
```

Запуск в watch-режиме:

```bash
npm run test:watch
```

## Симуляция RTP (CLI)

```bash
npm run simulate:rtp
```

По умолчанию: `100000` спинов, сид `42`, ставка `10`. Свои значения:

```bash
npm run simulate:rtp -- 50000 123 5
```

Аргументы: `спины`, `seed`, `ставка`.

| Слой        | Технологии                          |

| Язык       | TypeScript (strict)                  |
| Тесты      | Vitest                               |

## Структура `src/`

- **`app/`** — bootstrap, конфиг приложения и UI
- **`shared/`** — общие константы и утилиты
- **`game/application/`** — контроллеры, стор, события, FSM
- **`game/domain/`** — правила слота, движки выплат, барабанов
- **`game/config/`** — полосы, пейтабл, баланс, бонус
- **`game/math/`** — RNG, симуляции RTP
- **`game/presentation/`** — сцены, Pixi-компоненты, ассеты (пиксельные текстуры через Canvas)

Символы по умолчанию рисуются в коде (`presentation/assets/pixelSymbolTextures.ts`), отдельные PNG в репозитории не обязательны.
